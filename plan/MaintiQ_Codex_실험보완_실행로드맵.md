## 우선순위

가장 먼저 현재 0.86 결과와 테스트를 고정하고 공통 실험 기록 기능을 만든다. 그다음 반복 층화 교차검증·부트스트랩과 한 번 실행 재현 스크립트를 구현한다. 이후 실제 시간축 SPC, 생성형 AI 다중 사례 평가, SCANIA 제약 정책, 실제 현장 검증 순으로 진행한다.

## 항목별 핵심

1. **현장 검증**
   실제 회사 데이터가 없으면 스키마·섀도 모드 준비 상태만 검증한다. 실제 자료가 제공되면 시간 순서로 학습·정책 선택·섀도 테스트를 분리하고 이벤트 재현율, 설비일당 오경보, 리드타임, 교대당 경보량을 평가한다. 비용·다운타임 절감은 완전한 직접 기록이 있을 때만 계산한다.

2. **실제 시간축 SPC**
   MetroPT-3로 정상 기준기간과 이후 고장기간을 시간 순서로 분리한다. Shewhart, EWMA, 위험점수 관리한계를 비교하고 고장 이벤트 탐지율, 리드타임, 하루 오경보, 경보 지속시간을 산출한다.

3. **생성형 AI**
   저위험·경계·고위험·불완전 입력을 포함한 사례 파일을 만든다. 숫자·센서명 일치, 원인 단정, 자동 정비 명령, 사람 승인 문구를 자동 검사한다. 실제 전문가가 없으면 전문가 점수를 만들지 않는다.

4. **SCANIA**
   전부 경보를 실패 기준선으로 보존한다. 학습 자료 안에서 경보율 제약을 선택한 뒤 공식 검증 자료에는 한 번만 적용한다. 비용, Macro-F1, 균형정확도, 클래스별 재현율, 경보율을 함께 보고한다.

5. **강건성 검증**
   현재 코드가 seed마다 모델을 다시 학습하고 검증 임계값을 다시 선택하는지 테스트로 고정한다. 이후 5회 반복 5-fold 외부 검증과 내부 임계값 선택을 수행하고 out-of-fold 예측에 2,000회 부트스트랩 95% 구간을 계산한다.

6. **초기 80:20 결과**
   새 분석이 승인된 후 표 5.2와 그림 5.4를 부록으로 옮긴다. 본문은 최종 60:20:20 결과만 중심으로 정리한다.

7. **선행연구**
   Codex는 원문 근거 CSV와 검증 도구를 만든다. 데이터셋, 모델, 검증, 지표, 설명, 생성형 AI, 승인·감사 이력, 한계를 기록하고 원문에 없는 값은 `NR`로 둔다.

8. **재현 자동화**
   quick 스크립트는 AI4I 기준 결과·테스트·앱 점검을 수행한다. full 스크립트는 선택적으로 강건성, MetroPT-3, SCANIA, GenAI 오프라인 평가를 실행한다. 각 실행은 커밋·환경·데이터·명령·산출물 해시를 별도 폴더에 남긴다.
