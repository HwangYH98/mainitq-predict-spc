# MaintiQ Predict Samples

`company_sensor_sample.csv` is a small company-style CSV for checking the desktop prediction workflow.

Workflow:

1. Open MaintiQ Predict.
2. Go to the data prediction screen.
3. Load `samples/company_sensor_sample.csv`.
4. Review the automatic column mapping.
5. Run prediction.
6. Check failure probability, high-risk status, and risk-priority results.

The sample is derived from the AI4I-style feature schema and is not real company operating data.
